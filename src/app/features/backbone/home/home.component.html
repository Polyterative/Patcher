What is patcher.xyz?

This project solves the problem of transcribing and sharing patch notes to save them or simply to send them to someone. No more typing!

What can I do?

The patch builder allows connecting CV inputs and CV outputs in an easy and fast way.

You can then share the URL or save the list of connections as text. Patches are saved on the website itself and can be shared in with a simple link.

But doesn't this require having a list of all modules and their information?

Yes, however, we are not starting from scratch! Since one of the objectives is to maintain a database of inputs and outputs of each module, we ask for the help of the community to map all modules' CVs. We already have a lot of data mapped and ready.

What can I expect from this project in 3 months?

The mid-term goal is to create a Hub in which not only to share and create patches but also to manage your rack and all your modules in a manner similar to modular grid with the advantage of a more modern interface and above all more rapid navigation/editing and several other features yet to be announced!
